Instruction

Type color in "Color" text-field, line thickness in "Line" text-field and then press a button "Paint".
Color comes from the hex color palette for example #a020f0, line is greather than 0.
Typing commands use only spaces between functions.

Fuction

  • forward(length)
  • backward(length)
  • left(angle)
  • right(angle)
  • penDown()
  • penUp()
  • polygon(numberofsides)

Color

If you want, you can use a multicolor pen, then enter RANDOM in the "Color" field.
